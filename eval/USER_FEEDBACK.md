# User Feedback — AI20K‑162 (deliverable #10, phần Product)

> Mục tiêu: thu **≥3 phản hồi thật** từ người thử app → gỡ điểm "Product/Business".
> **KHÔNG bịa.** Điền sau khi thực sự cho người ta thử (mỗi người ~2 phút).

## Cách thu nhanh (2 phút/người)
1. Mở app (live URL hoặc `localhost:8000`) → để họ tự bấm **chip lệnh** hoặc gõ một mục tiêu tiếng Việt → bấm **Chạy thật** (hoặc **Demo nhanh**).
2. Cho xem **trace** (suy nghĩ → tool → kết quả thật) + tình huống **người chắn lối → replan**.
3. Hỏi 3 câu (bên dưới), ghi nguyên văn.

## 3 câu hỏi
- **Q1 (hiểu được không):** Bạn có hiểu agent đang làm gì qua bảng kế hoạch/trace không? (rõ/tạm/khó)
- **Q2 (giá trị):** Nếu là robot kho thật, tính năng nào hữu ích nhất? (ra lệnh tiếng Việt · tự replan · dừng/hỏi an toàn · audit log)
- **Q3 (tin được không):** Bạn có tin agent "không bịa" khi thấy mỗi bước đọc trạng thái thật không? Vì sao?
